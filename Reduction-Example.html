To show how optimal reduction works, we step through the example term M = o $ \h. o (h I) from pages 26-35 of [AG99]. To write the derivation tree we must define the recursive type \(\Omega = \, !\Omega \to \Omega\).

  Cut o1 o1l
    o1/[]/[] = PiRight ^func, [x_o1], [ret_o1], [], [] ->

      fp/[]/[] = Bang f []/[]
        f/[]/[] = PiRight ^func, [h_d], [f_ret], [], [] ->
          Cut o2 o2l
            o2/[]/[] = PiRight ^func, [x_o2], [ret_o2], [], [] ->
              x_o2 = BangC x1_o2 x2_o2

    )] [(ret_l, ret_l/ret = Identity)]
