# User Documentation

This folder contains the user documentation for Stroscot, organized according to the Diataxis framework. It is intended to help users learn and effectively use the Stroscot programming language.

**Purpose:** Help users learn and use the language/tool.

Folder structure (refer to Diataxis framework for more details):
- Tutorials (learning)
- How-To Guides (tasks)
- Reference (lookup)
- Explanations (understanding)
- Getting Started (onboarding, moved out from tutorials for visibility)
